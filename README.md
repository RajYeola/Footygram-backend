# Footygram-backend
Backend for the social media application [Footygram](https://github.com/RajYeola/Footygram)

## Tech Stack
- Express JS
- Mongo DB
- Mongoose

## Functionalities
- Posts
  - Add a new post
  - Delete post
  - Add reaction to post
  - Delete reaction from post

- User
  - Fetch all users
  - Follow other user
  - Unfollow other user
 
 - Authentication
   - Fetch user data
   - User signup
   - User login
